In this guide we will learn how to use the conformal coating inspection using the **AI-4050 AOI**.

!!! note "Note"
    This function requires install a separete hardware kit. To learn how to install the UV kit follow the instruction in the following section:
    [UV conformal coating installation guide](../maintenance/UV_kit_install.md)

!!! warning "Caution"
    The conformal coating inspection uses UV lighting so **we recommend to use the safety glasses** included with the kit.


    ![UV warning](../assets/v7/UV_inspection/warning_icon.png){width=200px, .center}

## 1. Generate a REFERENCE image or load it

To generate a **REFERENCE** image follow the steps in the [following guide](../how_to/Inspection_workflow.md#generating-a-reference) or [load a previous REFERENCE](../how_to/Screen-layout.md#load-reference-as-file).

## 2. Proceed with the inspection

Use the UUI capture buttom to open the [Live View window](../how_to/Inspection_workflow.md#capturing-an-uui) of the current PCBA to inspect.
In this window mark the UV inspection option located in the bottom side of the window.

![Live view window](../assets/v7/UV_inspection/UV_buttom.png){.center}
